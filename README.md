###### 部署

推荐使用docker部署，镜像：`待定`

###### 开发前的配置要求

1. 本项目采用`bun`，但你也可以采用其他包管理器(npm,yarn,pnpm,cnpm)

###### **安装步骤**

```sh
git clone https://github.com/skydog221/flintcore-edu.git
bun install
bun dev
```
